# randomBackgroundColor
## Описание
При нажатии кнопки, сайт случайным образом выдает цвет, который вы видите на экране и можете скопировать. Доступно по ссылке https://chubak-s.github.io/randomBackgroundColor/
## Технолгии
* HTML5
* CSS
* JS
* DOM
